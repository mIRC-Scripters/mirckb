/help
=====

The **/help** command opens up the mIRC manual to a specific topic. Consider using it as a last resort.

Synopsis
--------

.. code:: text

    /help [section]

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - [section]
      - a specific topic to lookup

Example
-------

.. code:: text

    /help /exit

Compatibility
-------------

Added: mIRC v2.1a (28 Feb 1995)
