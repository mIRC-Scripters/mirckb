/qmsg
=====

The /qmsg command sends the specified message to all open query windows.

Synopsis
--------

.. code:: text

    /qmsg <msg>

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <msg>
      - The message you want to send

Example
-------

None
