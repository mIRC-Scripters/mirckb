/vcrem
======

The **/vcrem** command can be used to remove a command from the voice command command list.

Synopsis
--------

.. code:: text

    /vcrem -ra <command1>[,command2[,etc..]]

Switches
--------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Switch
      - Description
    * - -r
      - removes the entry (default)
    * - -a
      - adds the entry

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <command1>
      - command to remove

Example
-------

None
