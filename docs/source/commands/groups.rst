/groups
=======

The **/groups** command displays a list of either all (no switch), enabled (-e), or disabled (-d) groups in your scripts.

Synopsis
--------

.. code:: text

    /groups [-e|d]

Switches
--------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Switch
      - Description
    * - -e
      - only enabled groups are displayed
    * - -d
      - only disabled groups are displayed

Parameters
----------

None
