/logview
========

The **/logview** command displays a text file in mIRC using the same method as the Log Files dialog.

Synopsis
--------

.. code:: text

    /logview -nN <filename>
 
Switches
--------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Switch
      - Description
    * - -nN
      - scrolls the opened file to the specified Nth line.

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <filename>
      - the filename you want to view
