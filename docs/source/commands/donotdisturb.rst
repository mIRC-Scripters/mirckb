/donotdisturb
=============

This allows you to temporarily disable all visible/audible notifications such as sounds, tips, flashing window icons, tray icon animations, and more.

Synopsis
--------

.. code:: text

    /donotdisturb on|off

Switches
--------

None

Parameters
----------

None

.. note:: Only works on Windows Vista and newer.

-------

None
