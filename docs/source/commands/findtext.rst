/findtext
=========

The **/findtext** command searches the active window for the specified text. The command default direction is up.

Synopsis
--------

.. code:: text

    /findtext -n <text>

Switches
--------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Switch
      - Description
    * - -n
      - look down

Parameters
----------

<text> the text to look for.

Example
-------

None

Compatibility
-------------

Added: mIRC v5.6 (23 Sep 1999)
