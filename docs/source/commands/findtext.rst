/findtext
=========

The **/findtext** command searches the active window for the specified text. The command default direction is up.

Synopsis
--------

.. code:: text

    /findtext -n <text>

Switches
--------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Switch
      - Description
    * - -n
      - look down

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <text>
      - the text to look for.

Example
-------

None
