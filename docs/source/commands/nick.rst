/nick
=====

The /nick command changes your nickname.

Synopsis
--------

.. code:: text

	/nick <nickname>

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <nickname> 
      - The new nickname

Example
-------

.. code:: text

	/nick foo
