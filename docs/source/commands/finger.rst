/finger
=======

The **/finger** command fingers a nickname.

Synopsis
--------

.. code:: text

   /finger <nick>

Switches
--------

None

Parameters
----------

.. list-table::
   :widths: 15 85
   :header-rows: 1

   * - Parameter
     - Description
   * - <nickname>
     - the nickname you want to finger
