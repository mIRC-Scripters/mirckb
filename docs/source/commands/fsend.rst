/fsend
======

The **/fsend** command turns on/off the fast dcc send feature. Displays a line with the current status (active window if applicable, status window otherwise).

Synopsis
--------

.. code:: text

    /fsend [on|off]

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - [on | off ]
      - if specified, enable/disable the fast send option
