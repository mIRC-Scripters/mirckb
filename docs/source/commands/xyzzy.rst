/xyzzy
======

The /xyzzy command prints out "Nothing happens". This is an easter egg.

Synopsis
--------

.. code:: text

    /xyzzy

Switches
--------

None

Parameters
----------

None

Example
-------

.. code:: text

    /xyzzy
    
    The above example outputs:
    Nothing happens.
    ... using this echo command:
    //echo $color(ctcp) -a Nothing happens.

-------------
