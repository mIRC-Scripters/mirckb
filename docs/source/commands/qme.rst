/qme
====

The /qme command sends the specified message as an action to all open query windows.

Synopsis
--------

.. code:: text

    /qme <msg>

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <msg>
      - The message you want to send

Example
-------

None

-------------
