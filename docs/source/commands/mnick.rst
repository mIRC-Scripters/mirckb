/mnick
======

The /mnick command changes the client's main nickname.

Synopsis
--------

.. code:: text

    /mnick <nickname>

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - <nickname>
      - the nickname you want to use
