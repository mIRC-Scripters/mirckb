/quit
=====

The /quit command disconnects you from IRC and will give the optional message as the reason for your departure.

Synopsis
--------

.. code:: text

    /quit [msg]

Switches
--------

None

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - [msg]
      - The optional quit message

Example
-------

None
