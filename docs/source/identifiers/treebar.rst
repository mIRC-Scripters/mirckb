$treebar
========

$treebar gives you the current state of the mIRC treebar.

Synopsis
--------

.. code:: text

    $treebar

Parameters
----------

None

Properties
----------

None

Results
-------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Value
      - Description
    * - 0
      - The treebar is currently off.
    * - 1
      - The treebar is currently on.

Example
-------

Echo the treebar status to the active window:

.. code:: text

    //echo -a $treebar
