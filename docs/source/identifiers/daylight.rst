$daylight
=========

The $daylight identifier will return the seconds offset for daylight savings. If daylight savings is not in effect, returns $daylight 0.

Synopsis
--------

.. code:: text

    $daylight

Parameters
----------

None

Example
-------

Echo the daylight savings offset to the active window:

.. code:: text

    //echo -a $daylight
