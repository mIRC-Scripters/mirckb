$dccport
========

The $dccport identifier returns the current port number being used by the /dccserver command - mIRC|DCCSERVER.

Synopsis
--------

.. code:: text

    $dccport

Parameters
----------

None

Example
-------

Echo to the active screen the port being used by the /dccserver command - mIRC|DCCSERVER:

.. code:: text

    //echo -a $dccport

-------------
