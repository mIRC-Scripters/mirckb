$mircdir
========

$mircdir return the folder in which mIRC stores its main settings files such as mirc.ini.

Synopsis
--------

.. code:: text

    $mircdir

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $mircdir
