$mircpid
========

$mircpid returns the process ID for the current executable

Synopsis
--------

.. code:: text

    $mircpid

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a the process ID for this instance of mIRC is $mircpid
