$ebeeps
=======

$ebeeps returns either :doc:`$true </identifiers/true>` or :doc:`$false </identifiers/false>` depending on whether or not sounds are enabled in the options dialog.

Synopsis
--------

.. code:: text

    $ebeeps

Parameters
----------

None

Example
-------

Echo whether or not sounds are enabled:

.. code:: text

    //echo -a $ebeeps

-------------

.. compatibility:: 6.3
