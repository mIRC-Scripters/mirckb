$r
==

 $r is a shortcut for :doc:`$rand </identifiers/rand>`.

Synopsis
--------

.. code:: text

    $r(<v1>,<v2>)

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - v1, v2
      - Start and end of the range of values (numbers or letters) to be returned by $r.
