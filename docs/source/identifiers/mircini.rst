$mircini
========

$mircini return the complete filename to the mirc.ini setting file currently used by the application.

Synopsis
--------

.. code:: text

    $mircini

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $mircini
