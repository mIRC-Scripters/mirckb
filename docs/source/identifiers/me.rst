$me
===

$me returns your current nickname.

Synopsis
--------

.. code:: text

    $me

Switches
--------

None

Results
-------

Returns the nickname that you are currently using.

Example
-------

Echo your current nickname to the active window

.. code:: text

    //echo -a $me
