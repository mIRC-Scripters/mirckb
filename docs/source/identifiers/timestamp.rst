$timestamp
==========

$timestamp returns the current time based on the event timestamp format as defined in the mIRC options.

Synopsis
--------

.. code:: text

    $timestamp

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $timestamp
