$status
=======

$status returns the server connection status, can be "connecting", "connected", "disconnecting", "closing" or "disconnected".

Synopsis
--------

.. code:: text

    $status

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $status
