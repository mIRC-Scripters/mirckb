$snicks
=======

$snicks returns a string of the currently selected nicknames in the active channel listbox, comma separated.

Synopsis
--------

.. code:: text

    $snicks

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $snicks

-------------
