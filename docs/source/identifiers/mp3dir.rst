$mp3dir
=======

.. attention:: This feature has essentially been replaced by :doc:`$sound </identifiers/sound>`

$mp3dir returns the directory specified in the Sound Requests section of the Options dialog for mp3 files.

Synopsis
--------

.. code:: text

    $mp3dir

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $mp3dir

-------------

.. compatibility:: 5.8
