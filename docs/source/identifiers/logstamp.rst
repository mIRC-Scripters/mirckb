$logstamp
=========

$logstamp returns the current time based on the logging timestamp format as defined in the Options.

Synopsis
--------

.. code:: text

    $logstamp

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $logstamp
