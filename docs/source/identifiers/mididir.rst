$mididir
========

.. attention:: This feature has essentially been replaced by :doc:`$sound </identifiers/sound>`

$mididir returns the directory specified in the Sound Requests section of the Options dialog for midi files

Synopsis
--------

.. code:: text

    $mididir

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $mididir

-------------
