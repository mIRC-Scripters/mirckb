$inmode
=======

$inmode is a value that can be returned by $chan().banlist if mIRC is currently in a mode +b listing.

Synopsis
--------

.. code:: text

    $inmode

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $chan(1).banlist
