$time
=====

$time returns the current time in hour:minute:second format. The time is returned in 24-hour clock format, not 12-hour clock format.

Synopsis
--------

.. code:: text

    $time

Parameters
----------

None

Properties
----------

None

Examples
--------

Echo the current time to the active window:

.. code:: text

    //echo -a $time

-------------
