$version
========

$version retrieves the current version of mIRC that you are using.

Synopsis
--------

.. code:: text

    $version

Parameters
----------

None

Properties
----------

None

Results
-------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Value
      - Description
    * - vN.NN
      - vN.NN would be replaced with the actual mIRC version that mIRC returns to you.

Example
-------

Echo the current mIRC version to the active window:

.. code:: text

    //echo -a $version
