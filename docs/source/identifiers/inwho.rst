$inwho
======

$inwho is a value returned by :doc:`$chan </identifiers/chan>` or $chan().inwho if mIRC is currently handling a /who reply and this is the first /who.

Synopsis
--------

.. code:: text

    $inwho

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $chan().inwho

-------------
