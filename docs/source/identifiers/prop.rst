$prop
=====

$prop returns the name of the property passed to your custom alias, if any.

Synopsis
--------

.. code:: text

    $prop

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    alias test {
      echo -a $prop
    }
//noop $test().prop
