$ssldll
=======

$ssldll returns the filename to the ssl dll used by mIRC. In newer versions of mIRC, SSL support is integrated and the :doc:`$mircexe </identifiers/mircexe>` path will be returned.

Synopsis
--------

.. code:: text

    $ssldll

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $ssldll

-------------
