$trust
======

$trust returns items in the dcc trust list.

Synopsis
--------

.. code:: text

    $trust(N)

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - N
      - The Nth item in the dcc trust list, if N = 0, returns the total number of item in the list

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $trust(0)

-------------
