$awaymsg
========

$awaymsg will return the message you set up when you originally used the /away command - mIRC|/away command.

Synopsis
--------

.. code:: text

    $awaymsg

Parameters
----------

None

Example
-------

Echo your away msg to the active window

.. code:: text

    //echo -a $awaymsg

-------------
