$away
=====

$away will return ''$true'' if you are currently away, otherwise it will return ''$false''.

Synopsis
--------

.. code:: text

    $away

Parameters
----------

None

Example
-------

Echo away status to the active window

.. code:: text

    //echo -a $away

-------------
