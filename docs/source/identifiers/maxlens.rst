$maxlens
========

$maxlens return the maximum number of characters that can safely be used for small string in a script.

The current value is 512

Synopsis
--------

.. code:: text

    $maxlens

Parameters
----------

None

Examples
--------

.. code:: text

    //echo -a $maxlens
