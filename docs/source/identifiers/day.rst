$day
====

The $day identifier returns the full name of the current day of the week, eg: Monday, Tuesday, etc.

Synopsis
--------

.. code:: text

    $day

Parameters
----------

None

Example
-------

Echo the current day to the active window:

.. code:: text

    //echo -a $day
