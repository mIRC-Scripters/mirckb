$scriptdir
==========

$scriptdir returns the folder containing the current script file, or if used in an editbox, return the default folder for script.

Synopsis
--------

.. code:: text

    $scriptdir

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $scriptdir
