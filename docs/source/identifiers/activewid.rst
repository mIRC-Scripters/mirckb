$activewid
==========

$activewid returns the window ID for the currently active window.

Synopsis
--------

.. code:: text

    $activewid

Parameters
----------

None

Example
-------

Echo the current window's id

.. code:: text

    //echo -a $activewid

-------------

.. compatibility:: 6.2
