$true
=====

$true is the result of a successful comparison operation and various other identifiers.

Synopsis
--------

.. code:: text

    $true

Parameters
----------

None

Properties
----------

None

Example
-------

Echo to the active window a result that returns $true:

.. code:: text

    //echo -a $isfile($mircexe)
