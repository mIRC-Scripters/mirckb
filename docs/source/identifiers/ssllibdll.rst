$ssllibdll
==========

$ssllibdll returns the complete filename to the lib used for SSL. In recent versions of mIRC, SSL is integrated with mIRC and the :doc:`$mircexe </identifiers/mircexe>` value will be returned.

Synopsis
--------

.. code:: text

    $ssllibdll

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $ssllibdll

-------------
