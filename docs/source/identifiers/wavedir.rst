$wavedir
========

.. attention:: This feature has essentially been replaced by :doc:`$sound </identifiers/sound>`

$wavedir returns the directory specified in the Sound Requests section of the Options dialog for wav files.

Synopsis
--------

.. code:: text

    $wavedir

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $wavedir

-------------
