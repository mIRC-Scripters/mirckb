$active
=======

$active will return the full name of the currently active window.

Synopsis
--------

.. code:: text

    $active

Parameters
----------

None

Example
-------

Echo the value of $active to the active window

.. code:: text

    //echo -a $active

-------------

.. compatibility:: 4.5
