$bits
=====

$bits returns the bit version for the mIRC application you are using; either 32 or 16.

Synopsis
--------

.. code:: text

    $bits

Parameters
----------

None

Example
-------

Echo mIRC's application bit number to the active window

.. code:: text

    //echo -a $bits
