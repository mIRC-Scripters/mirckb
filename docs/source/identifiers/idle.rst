$idle
=====

$idle return your current idle time, same idle time as that returned by ctcp USERINFO or FINGER.

Synopsis
--------

.. code:: text

    $idle

Parameters
----------

None

Properties
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Property
      - Description
NOne

Example
-------

.. code:: text

    //echo -a $idle
