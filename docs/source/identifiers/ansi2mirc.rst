$ansi2mirc
==========

$ansi2mirc converts ANSI color codes into mIRC-supported Colors - IRC|color codes.

Synopsis
--------

.. code:: text

    $ansi2mirc(<ANSI>)

Properties
----------

None

Example
-------

Echo some ANSI code, converted, to the active window

.. code:: text

    //echo -ga $ansi2mirc(this is probably $chr(27) $+ [32mgreen $+ $chr(27) $+ [0m.)

Compatibility
-------------

.. compatibility:: 5.5
