$mnick
======

$mnick returns the value of your main nickname, which is set in the mIRC options dialog.

Synopsis
--------

.. code:: text

    $mnick

Switches
--------

None

Results
-------

Returns the value of your main nickname.

Example
-------

Echo the value of $mnick to the active window:

.. code:: text

    //echo -a $mnick
