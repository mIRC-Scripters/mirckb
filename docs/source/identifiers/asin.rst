$asin
=====

$asin return the arcsine of N.

Synopsis
--------

.. code:: text

    $asin(N)[.deg]

Parameters
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Parameter
      - Description
    * - N
      - The number you want the arc sine of, in radian.

Properties
----------

.. list-table::
    :widths: 15 85
    :header-rows: 1

    * - Property
      - Description
    * - .deg
      - Returns the angle as degrees, default is radians.

Example
-------

.. code:: text

    //echo -a $asin(5)

-------------
