$fullscreen
===========

$fullscreen returns 1 if an application is currently running in full screen mode, 0 if not. This applies to both game and non-game applications.

Synopsis
--------

.. code:: text

    $fullscreen

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $fullscreen

-------------
