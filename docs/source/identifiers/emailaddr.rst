$emailaddr
==========

$emailaddr returns the email address you have specified in the mIRC connect dialog.

Synopsis
--------

.. code:: text

    $emailaddr

Switches
--------

None

Example
-------

Echo the email address you provided to the currently active window

.. code:: text

    //echo -a $emailaddr
