$switchbar
==========

$switchbar returns :doc:`$true </identifiers/true>` if the switchbar is enabled in mIRC, :doc:`$false </identifiers/false>` otherwise.

Synopsis
--------

.. code:: text

    $switchbar

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $switchbar
