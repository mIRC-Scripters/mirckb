$timestampfmt
=============

$timestampfmt returns the current timestamp format as defined in the mIRC options.

Synopsis
--------

.. code:: text

    $timestampfmt

Parameters
----------

None

Properties
----------

None

Example

    //echo -a $timestampfmt
