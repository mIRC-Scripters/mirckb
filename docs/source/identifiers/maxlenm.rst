$maxlenm
========

$maxlenm return the maximum number of characters that can safely be used for medium string in a script.

The current value is 2048

Synopsis
--------

.. code:: text

    $maxlenm

Parameters
----------

None

Examples
--------

.. code:: text

    //echo -a $maxlenm
