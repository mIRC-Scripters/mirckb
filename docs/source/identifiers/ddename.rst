$ddename
========

$ddename return the name of the current DDE service in use by the mIRC DDE Server.

Synopsis
--------

.. code:: text

    $ddename

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $ddename
