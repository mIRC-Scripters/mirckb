$dlevel
=======

$dlevel will return the current default user level.

Synopsis
--------

.. code:: text

    $dlevel

Parameters
----------

None

Example
-------

Echo the value of $dlevel to the active window:

.. code:: text

    //echo -a Default user level: $dlevel

-------------
