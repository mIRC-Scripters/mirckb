$titlebar
=========

$titlebar returns the current mIRC titlebar.

.. note:: Use :doc:`$window </identifiers/window>`(window).title to get a custom window titlebar.

Synopsis
--------

.. code:: text

    $titlebar

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $titlebar

-------------
