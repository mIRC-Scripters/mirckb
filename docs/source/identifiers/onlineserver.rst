$onlineserver
=============

$onlineserver returns the current server online time, same as :doc:`$online </identifiers/online>`.

Synopsis
--------

.. code:: text

    $onlineserver

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $onlineserver

-------------
