$email
======

$email returns the email address you have specified in the mIRC connect dialog.

Synopsis
--------

.. code:: text

    $email

Switches
--------

None

Example
-------

Echo the email address you provided to the currently active window

.. code:: text

    //echo -a $email

-------------
