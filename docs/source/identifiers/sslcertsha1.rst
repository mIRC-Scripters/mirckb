$sslcertsha1
============

$sslcertsha1 returns the sha1 fingerprint of the client certificate in the currently loaded private key file

Synopsis
--------

.. code:: text

    $sslcertsha1

Parameters
----------

None

Properties

    //echo -a $sslcertsha1
