$lactivewid
===========

$lactivewid returns the window ID for the last active window.

Synopsis
--------

.. code:: text

    $lactivewid

Parameters
----------

None

Example
-------

Echo the last active window's id

.. code:: text

    //echo -a $lactivewid

-------------
