$nickmode
=========

$nickmode returns the list of nick modes that are supported on a server in letter form, or a default value of "ohv" if not connected.

Synopsis
--------

.. code:: text

    $nickmode

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $nickmode
