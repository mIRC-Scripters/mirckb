$anick
======

$anick returns the value of your alternative nickname, which is set in the mIRC options dialog.

Synopsis
--------

.. code:: text

    $anick

Switches
--------

None

Results
-------

Returns the value of your alternative nickname.

Example
-------

Echo the value of ``$anick`` to the active window

.. code:: text

    //echo -a $anick
