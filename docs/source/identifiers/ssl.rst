$ssl
====

$ssl returns :doc:`$true </identifiers/true>` if you are connected with SSL to a server, :doc:`$false </identifiers/false>` otherwise.

Synopsis
--------

.. code:: text

    $ssl

Parameters
----------

None

Properties
----------

None

Example

    //echo -a $ssl
