$servertarget
=============

$servertarget returns the original address used to connect to the IRC server.

This can be useful when dealing with BNC or Twitch.

Synopsis
--------

.. code:: text

    $servertarget

Parameters
----------

None

Properties
----------

None

Example
-------

.. code:: text

    //echo -a $servertarget
